Comando bÃ¡sicos de git bash
    cd => te lleva a la carpeta principal del usuario (Ej: /c/Users/Pedro).
    cd . =>te mantiene en la carpeta actual.
    cd .. => te regresa al directorio padre.
    cd (nombre_directorio) => ingresas al directorio ingresado.
    ls => muestra el contenido del directorio.
    pwd => muestra la ruta del directorio en el que te encuentras.
    code . => abre vsCode en el directorio actual
    git config => te muestra las opciones de configuracion de git bash.
    git init => iniciliza un repositorio.
    git status => muestra el estado del repositorio actual.
    git add (nombre_archivo) => agrega el archivo al commit.
    git add *.(extension_deseada) => agrega todos los archivos con la extension elegida al commit.
    git add . => agrega todos los archivos del directorio al commit.
    git commit -m "Mensaje describiendo el commit" => se realiza el commit y se agrega una descripcion del mismo
